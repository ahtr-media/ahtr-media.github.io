このウェブサイトは、まちづくりの現場で生まれる「葛藤」を記録・発信し、社会的資源として共有・蓄積していくことを目指す研究プロジェクトの記述プラットフォームです。



### まちづくりにおける「葛藤」について



地域のまちづくりでは、年齢や立場、価値観のちがうたくさんの人が同じ場所のことを考えます。ある人にとって大切なことが、別の人には困ることになる、ということも珍しくありません。多様な価値観をもつ人びとが集まりひとつの街について考えることは、しばしばそうした意見のぶつかり合いや対立を生み、「葛藤」が芽生えます。あるいは、そうした葛藤が表面化されない、まだ言葉にもなっていないモヤモヤが、地域の関係性に影を落とすこともあるでしょう。



まちづくりとは、住民が主体となって地域を共同で利用・管理していく営みであり、その根本的な課題は「異なる価値観の共存」にあると言われてきました。多様な価値観が交差するまちづくりの現場では、その街ごとに固有の出来事や葛藤が生まれます。街に関わる人びとの合意形成を目指すまちづくりの実践のなかでは、葛藤は解消すべきもの、乗り越えるべきものとして扱われがちです。しかし、そもそも葛藤が生まれる場とは、異なる価値観が同じ場所で実際に出会っている状況にほかなりません。そうした出会いは、地域に新たな価値を生みだす契機にもなり得るのです。



だからこそ、葛藤を「多様性の時代」の社会における資源として位置づけなおし、まちづくりの当事者が地域に生まれる葛藤を新たな価値創造のきっかけとして創造的に受けとめられるよう、葛藤や、まだ言葉になっていないモヤモヤを丁寧に記述し、共有することが重要だと、わたしたちは考えています。



### 「葛藤」を記述すること、問いなおすこと、自分自身を更新すること



このウェブサイトでは、建築・都市計画・社会心理学・文化人類学・社会学などの分野でまちづくりの現場に関わる研究者たちが、それぞれの現場での葛藤を記述し、その経験を持ち寄り、問いなおしを重ねる過程を発信・共有することを目指しています。



葛藤の事例や普遍的に適用できる解決策・方法論を提示することが、このウェブサイトの目的ではありません。



記述するだけでなく、それをどんな場で共有するか、どんな表現で伝えるか、といった「場所」や「媒体」の工夫も、葛藤を社会に開いていくうえで欠かせない要素です。写真や音・映像、デジタル技術を活用した記録など、さまざまな実験的な表現を試みながら、現場に直接関わらない人びととも葛藤を広く共有できる媒体のあり方を探り、発信しながら検証をつづけます。



また、記述して終わりでもありません。記述が読まれ、問いなおされ、考えが更新されていくことが、葛藤の創造性を示すはずです。数年後に「あのとき自分はこう思っていたけれど、いまは少しちがう」という気づきが生まれることもあるでしょう。考えが変わっていく過程を読者が追うことで得られる気づきもあるでしょう。そうした継続的な書き直しの可能性——「訂正できること」を、わたしたちは大切にしています。記事は公開後もつねに更新される可能性に開かれています。



わたしたちはいつも、わたしたち自身とわたしたちの街をわかっている途中なのです。



### ウェブサイトの仕組みについて



こうした考えに基づいてウェブサイトを運営するとき、ウェブサイトの仕組み自体においても同じ思想でつくられていることが重要だと、わたしたちは考えました。何を企画し、どう編集し、何を伝えて、何を残さないか——その選択の過程には、現場の葛藤と同じような問いが生じます。制作する側が情報を一方的に「管理」するのではなく、その過程をできるだけ開いておくことが、つくり手と読み手のあいだにも、新しい対話を生むのではないかと考えています。



具体的には、記事がどのように変わってきたかという編集の履歴を残し、公開し、誰でも確認できるようにしています。「なかったことにする」のではなく、書きなおしの記録が残ること——つねに更新される可能性を開くと同時に、更新の前後を公開することで、書きなおしにも意味があることを示せると考えています。



こうした仕組みを実現し、また限られた予算のなかで持続的に運営するために、このウェブサイトはソースコード管理・公開サービスであるGitHub / GitHub Pagesを活用しています。これにより、編集履歴の公開と、長期的な透明性・持続可能性を確保しています。

### 本研究について詳しく知る（アコーディオン表示）



本研究の正式名称は「地域のまちづくりにおける葛藤の創造的受容と記述プラットフォームの構築」（科学研究費補助金 基盤研究（B）、2025〜2028年度、研究代表者：前田昌弘・京都大学 人間・環境学研究科）。建築・都市計画を中心に、社会心理学・文化人類学・社会学などの隣接領域の研究者が協働する学際的研究です。



「葛藤」は、ただちに調停されない価値の対立として位置づけられます。同時に、そもそも言葉にすらなっていない「葛藤未満」の状態——記述されないまま消えていくモヤモヤ——にも、このプロジェクトは向き合います。気づきを持つこと、他者の価値観と接すること自体が、記述と共有の出発点です。



前段の科研（「社会的葛藤を抱える地域のまちづくりにおける価値の公正な記述手法の開発」、基盤（B）23K20991、2021〜24年度）では、複数の主体が同じフィールドを異なるレベルで記述する重層性と、記述が当事者に返り再び記述を生む再帰性を組み込んだアプローチの有効性が示されました。同時に、記述を共有する「場所」（どのような状況・場面で分かち合うか）や「媒体」（言葉・文章にとどまらず、ビジュアルや音・映像などの表現を含む）の重要性が明らかになり、これらを含めた「記述の拡張」が次の課題として残されました。本研究はこれを受け、現場での葛藤の記述とその拡張としての場所・媒体からなる体系を「記述プラットフォーム」と名づけ、その構築・実践・検証を進めます。協働・災害・風景・居場所・異文化など、テーマと地域の異なる複数のフィールドで実践的研究を重ね、年次で対話の場を設け、成果を社会に開いていきます。



コンテンツは単なる事例紹介にとどまらず、「レビュー（批評・問いなおし）」の重要性を前面化したメディア実践として位置づけます。可逆性（なかったことにすること）ではなく訂正可能性（書き直しの記録を残すこと）の担保が、記述における誠実さだと考えています。



ウェブサイトの技術的な基盤として、本サイトはGitおよびGitHub / GitHub Pagesを採用しています。Gitとは、「いつ・誰が・何を変更したか」をすべて記録する分散型バージョン管理システムです。GitHubはそのGitをベースにしたソースコード管理・公開プラットフォームであり、GitHub Pagesはそのリポジトリから直接ウェブサイトを公開できる静的サイトホスティングサービスです。パブリックリポジトリであれば無料で利用でき、独自ドメインを使わない場合（`xxxxx.github.io`形式）はドメイン費用も発生しません。すべてのコンテンツはファイルとしてリポジトリに保存されており、ZIPファイルとしてバックアップ・持ち出しが可能です。CMSに依存しない静的HTMLによる運用のため、サーバー環境への依存を最小限に抑えられます。仮にGitHubがサービスを終了しても、バックアップさえあれば別の場所でほぼそのまま復元できます。
